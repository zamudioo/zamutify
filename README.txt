Open the playlist_urls.txt file and add the URLs of the YouTube playlists you want to download. Each URL should be on a separate line.

In the metadatos.py file, make sure to properly configure the folder_path on line 35. This is the location of your MP3 files that you want to update with metadata.

In the download_playlists.bat file, make sure to properly configure the playlist URLs in the PLAYLIST_URLS variable on line 8.

Running the Project

Once you have completed the setup, you can run the project by executing the file named "script.bat"
